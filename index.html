How to submit
  1. Format Your Submission
    • Paste your text into the Content box on the left
    • Fill out all required fields
    • Click Copy All
  2. Upload to GitHub
    • Navigate to the content/ folder in the repository
    • Open the folder where your article should go
    • Click Add file → Create new file
  3. Add Your File
    • Paste the copied content into the new file
    • Name the file using the title of your page (.md format)
  4. Commit Your Changes
    • Scroll down and click Commit changes
